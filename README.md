# Student Management System

## Description
A Python project for managing student information.
The system allows users to add, view, search, update, and delete student records.

## Features
- Add new students
- Display all students
- Search for students
- Update student information
- Delete student records

## Technologies
- Python

## Project Structure

Student-Management-System/
│
├── main.py
├── students.json
└── README.md

## How to Run

1. Install Python.
2. Open the project folder.
3. Run:

python main.py
